
# TapCanvas

**可视化 AI 创作画布（零 GPU）**
用第三方 AI API（Sora / 可灵 / 即梦 / 图像生成 / TTS 等）生成多媒体素材；前台以 **React-Flow** 搭建“节点/连线”工作流画布，配合 **Remotion** 做时间线预览与导出；后台使用 **Activepieces** 作为轻量编排层，负责提交任务、轮询/回调、重试/限流、资产落盘（S3/OSS）与通知前端。

## ✨ 核心特性

- **零 GPU**：全部生成走第三方 AI API。
- **所见即所得的画布**：React-Flow 节点化编辑，支持分镜/镜头参数与模板。
- **时间线预览与导出**：Remotion 组件化时间线，预览流畅，导出一键触发。
- **稳健编排**：Activepieces 处理 Webhook、异步任务、重试/退避、幂等与速率限制。
- **资产治理**：自动落盘至 S3/OSS，记录 `artifacts/tasks` 元数据与成本信息。
- **可扩展节点**：轻松接入 Sora / 可灵 / 即梦 / 图像生成 / TTS / 字幕对齐 / 合成服务等。
